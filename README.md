# net11.cc Web Security Assessment (Educational Portfolio)

## Overview

The purpose of this project is to demonstrate practical experience with automated web security scanning, vulnerability analysis, and security reporting.

# Reported Findings

### 1. Potential Cross-Site Request Forgery (CSRF)

**Severity:** Medium

**Description:**
Burp Suite reported a potential Cross-Site Request Forgery (CSRF) issue during the automated assessment. The request appeared to lack sufficient anti-CSRF protections, which may allow unauthorized requests to be performed on behalf of authenticated users if additional conditions are met. Because this finding was generated by an automated scanner, manual verification is required to determine whether it represents an exploitable vulnerability or a false positive.

**Status:** Not Verified

---

### 2. Strict Transport Security Not Enforced

**Severity:** Low

**Description:**
Burp Suite reported that HTTP Strict Transport Security (HSTS) was not fully enforced for the application. Implementing HSTS helps protect users from protocol downgrade attacks and cookie hijacking by ensuring browsers communicate only over HTTPS. This observation should be reviewed and validated.

**Status:** Requires Review
